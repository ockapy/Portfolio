Projet SI pharamacie

Description

J'ai réalisé ce projet lors de mon BTS SIO, le but était de recentraliser toutes les informations utiles pour une pharamacie sur une seule application. L'application permet de gérer les stocks de médicaments, les ordonnances, les patients et les medecins. Ce projet à été réalisé en binôme avec un autre étudiant, nous avons utilisé ajax pour la communication entre le front-end et le back-end. Le back-end est développé en PHP et le front-end en HTML, CSS et JavaScript. Nous avons également utilisé une base de données MySQL pour stocker les données.

Retour d'expérience

Ce projet à été mon premier projet complet avec des technologies web, j'ai appris à travailler en équipe et à gérer un projet de A à Z. J'ai également appris à utiliser PHP et MySQL et la communication asynchrone avec ajax. Je suis allé à la rencontre de pharmaciens pour comprendre leurs besoins et leurs attentes. J'ai également appris à gérer les problèmes de sécurité liés à la gestion des données sensibles.
