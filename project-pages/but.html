Stage BUT - LIG

Description

Le but de ce projet était de tester et d'optimiser des algorithmes d'intelligence artificielle dans le but de programmer automatiquement un synthétiseur virtuel (VST). Le synthétiseur virtuel en question est le DX7, un synthétiseur FM très populaire dans les années 80. Le travail s'inspirait d'un projet de recherche sur la synthèse sonore et l'optimisation des paramètres de synthèse:
M. J. Yee-King, L. Fedden & M. d'Inverno, "Automatic Programming of VST Sound Synthesizers Using Deep Networks and Other Techniques"
J'ai été chargé de réutiliser le code de ce projet, de le faire fonctionner avec le VST du DX7 et de tester les différents algorithmes. De plus j'ai développer une script permettant la lecture et l’écriture de fichiers sysex, qui sont des fichiers de configuration pour le DX7.

Retour d'expérience

Le projet a été très intéressant, car il m'a permis de découvrir la synthèse sonore et la programmation de synthétiseurs virtuels ainsi que le développement IA. J'ai appris à utiliser le framework JUCE, qui est très populaire pour le développement audio. J'ai également appris à utiliser le langage de programmation C++ et Python et à travailler avec des fichiers binaires. J'ai rencontré quelques difficultés lors de la mise en place du projet, car le code était assez complexe et il y avait beaucoup de dépendances. Cependant, j'ai réussi à surmonter ces obstacles et à faire fonctionner le code. Quatres algorithmes d'optimisation ont été testés: l'algorithme Hill Climber, l'algorithme génétique, un réseau de neurones MLP (Multi Layer Perceptron) et un réseau Long Short-Term Memory (LSTM). A terme j'ai réussi à faire fonctionner le code et à obtenir des résultats intéressants en réduisant fortement les temps de calculs.
