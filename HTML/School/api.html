TP APIs

Ce projet a été réalisé en groupe de 2, le but était de mettre en place un site web qui utilise une api.

Le site utilise l'api de la base de données du jeu Monster Hunter World. La base de données de l'api n'est pas complète mais elle permet de récupérer des informations sur les monstres et les objet du jeu.
