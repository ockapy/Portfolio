Projet Pharmacie (BTS)

Ce projet a été réalisé en groupe de 2, le but était de mettre en place un site web en PHP qui permet de gérer une pharmacie.

Le site répond aux attentes des pharmaciens, il permet de gérer les stocks et les ordonnances des patients. avec une base de données sur les médicaments et les patients. c'est un intranet complet.
